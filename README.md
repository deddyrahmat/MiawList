# MiawList

MiawList adalah website untuk mencari tahu berbagai jenis kucing. Website ini dibangun menggunakan ReactJS. 
Data kucing di dapat dari open API yang bernama [The CAT API](https://documenter.getpostman.com/view/5578104/RWgqUxxh#19c0515e-27da-415a-b844-bc7fe37d038c). 
Background kucing bersumber dari [unsplash](https://unsplash.com/).

Api dihubungkan dengan menggunakan AXIOS.

Demo program bisa di lihat [disini](https://curious-basbousa-4cd2f5.netlify.app/)
